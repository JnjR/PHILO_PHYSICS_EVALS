Formulaire d'évaluation

Chaque évaluateur peut évaluer chaque présentateur une seule fois. L'auto-évaluation est bloquée. Jusqu'à 23 évaluateurs sont possibles (22 étudiants + Prof. Rousseau), mais comme un présentateur ne peut pas s'auto-évaluer, chaque présentateur reçoit au maximum 22 évaluations.

Export / Import

Sauvegarder, restaurer ou réinitialiser les données. Les données sont conservées localement dans le navigateur. L'importation JSON fusionne les fichiers (idéale pour rassembler les fichiers des étudiants) : les doublons présentateur/évaluateur sont ignorés automatiquement.
